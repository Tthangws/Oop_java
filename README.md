# 🚀 Hành Trình Luyện Tập Lập Trình Hướng Đối Tượng (Java OOP)

Kho lưu trữ (Repository) này là nơi mình lưu trữ toàn bộ các bài tập thực hành, mã nguồn mẫu và các dự án nhỏ trong quá trình học và luyện tập tư duy Lập trình Hướng đối tượng (OOP) với ngôn ngữ Java.

> 🛠️ **Trạng thái:** Dự án được cập nhật liên tục sau mỗi buổi học và làm bài tập.

---

## 🗺️ Các kiến thức và kỹ năng luyện tập

Dưới đây là danh sách các phần kiến thức cốt lõi của Java OOP mà mình đã và đang chinh phục. (Mục nào hoàn thành sẽ được đánh dấu `[x]`).

### 🔹 4 Tính chất cốt lõi của OOP
- [ ] **Tính Đóng Gói (Encapsulation):** Khai báo thuộc tính `private`, sử dụng các hàm `Getter/Setter` để bảo vệ và kiểm soát dữ liệu.
- [ ] **Tính Kế Thừa (Inheritance):** Tối ưu hóa tái sử dụng code bằng từ khóa `extends` giữa các lớp.
- [ ] **Tính Đa Hình (Polymorphism):** Luyện tập cơ chế Ghi đè phương thức (`Overriding`) và Nạp chồng phương thức (`Overloading`).
- [ ] **Tính Trừu Tượng (Abstraction):** Thiết kế cấu trúc hệ thống bằng Lớp trừu tượng (`Abstract Class`) và `Interface`.

### 🔹 Kỹ năng mở rộng & Xử lý logic Backend
- [ ] **Xử lý ngoại lệ (Exception Handling):** Kiểm soát lỗi chương trình bằng khối lệnh `try-catch-finally`, phân biệt Checked/Unchecked và tự định nghĩa `Custom Exception`.
- [ ] **Cấu trúc dữ liệu (Java Collections):** Làm việc với mảng động (`ArrayList`, `List`), `Set`, `Map` để quản lý danh sách đối tượng.
- [ ] **Thao tác dữ liệu nâng cao:** Đọc/Ghi dữ liệu từ File, kết nối cơ sở dữ liệu (JDBC), hoặc tối ưu hóa code bằng Stream API.

## 📂 Sơ đồ tổ chức thư mục (Project Structure)

Mã nguồn trong thư mục `src` được chia theo từng bài tập hoặc từng nhóm chủ đề kiến thức riêng biệt để dễ dàng quản lý và theo dõi:

```text
project/
├── src/                   # Thư mục chứa toàn bộ mã nguồn (.java)
│   ├── bai_tap_01/        # Thư mục con chứa bài tập số 1
│   ├── bai_tap_02/        # Thư mục con chứa bài tập số 2
│   └── ...                # Các bài tập tiếp theo được bổ sung dần vào đây
└── README.md              # File hướng dẫn và giới thiệu tổng quan này
